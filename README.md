# Hello world · Anchor

A [k256 Program OS](https://k256.xyz) starter — a minimal [Anchor](https://www.anchor-lang.com)
program with one instruction and a generated IDL.

```
programs/hello/src/lib.rs   # #[program] with an `initialize` instruction
programs/hello/Cargo.toml   # anchor-lang
Anchor.toml                 # program id + workspace config
```

## What Program OS does with it

1. **Build** — every push compiles this in an isolated Firecracker microVM,
   producing a verified `.so` (and an IDL when the toolchain emits one).
2. **Promote** — turn the artifact into a real program on chain: create a new
   program id (random or vanity) or upgrade an existing one. You sign with your
   own wallet or multisig — Program OS never holds your keys. Your real id is
   written into `declare_id!` at creation.
